# dokidoki-palavras

Código em C++ pra ajudar a conseguir o Good Ending em Doki Doki Literature Club.

Coisas que esse código faz:
- Consulta palavras individualmente e informa os gostos de cada uma.
- Fornece a lista completa de palavras de cada uma.
- Exibe tutorial meia-boca pro Good Ending.

Coisas que esse código não faz:
- Fornecer uma interface de gente, pq eu não sei fazer e não to com tempo de 
aprender.
- Devolver o tempo que PERDI alinhando os textos pra ficar minimamente apresentável.
- Evitar que vc chore no final do Good Ending.

O arquivo "palavrasdoki.txt" tem tudo em ordem alfabética. Portanto, se quiser que 
funcione bem, só não mexer. Bota os dois arquivos na mesma pasta e roda (duh).
